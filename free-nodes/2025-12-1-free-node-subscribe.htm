一、什么是科学上网？为何自己动手更香？

1.1 科学上网的定义

所谓“科学上网”，简单来说就是通过一定的技术手段突破网络封锁和访问限制，从而连接至境外互联网资源。常见的技术手段包括VPN、代理服务器、VPS搭建Shadowsocks/V2Ray服务等。

它不仅能帮助用户获取被屏蔽的网站，还能实现更安全的数据传输、屏蔽追踪器，提升网络体验。

1.2 为什么选择自己架设？

虽然付费VPN服务看似简单方便，但却存在以下隐忧：

  • 安全性低：你的数据可能会被第三方记录、出售或监控

  • 价格昂贵：长期订阅高质量VPN动辄上百元每年

  • 灵活性差：节点固定、协议单一，无法满足个性化需求

  • 频繁掉线：节点经常不稳定或被封锁

相比之下，自建代理服务的优势非常明显：

  • ✅ 完全掌控：你控制一切数据流通，无后顾之忧

  • ✅ 性价比高：每月只需5美元左右，性价比远高于VPN

  • ✅ 自由定制：选择任意服务器、端口、协议、加密方式

  • ✅ 提升技术能力：学会了这一技能，未来可快速迁移或扩展

  • Trojan / Hysteria：更注重隐蔽性和抗封锁能力

本文以Shadowsocks为例，但你也可拓展学习V2Ray或Trojan的使用。

三、详细搭建流程（以Shadowsocks为例）

3.1 第一步：连接到你的VPS

在终端中输入：

bash
ssh root@your_vps_ip

然后输入你的VPS密码，成功连接后即可操作。

3.2 第二步：更新系统并安装依赖

执行以下命令：

bash
sudo apt update && sudo apt upgrade -y
sudo apt install -y python3-pip git curl

这会确保你的系统和依赖都是最新的，减少后续出错可能。

3.3 第三步：安装Shadowsocks服务端

使用 pip 安装 Shadowsocks：

bash
pip3 install https://github.com/shadowsocks/shadowsocks/archive/master.zip

3.4 第四步：创建配置文件

创建配置文件路径 /etc/shadowsocks.json，输入如下内容：

json
{
    "server": "0.0.0.0",
    "server_port": 8388,
    "password": "your_password_here",
    "timeout": 300,
    "method": "aes-256-gcm"
}

提示：建议使用复杂密码，并选择加密方式如 aes-256-gcm 或 chacha20-ietf-poly1305。

3.5 第五步：启动Shadowsocks服务

bash
ssserver -c /etc/shadowsocks.json -d start

你也可以将其配置为开机自启，或使用 systemd 管理进程。

四、本地配置与测试连接

4.1 下载Shadowsocks客户端

根据你的设备选择合适客户端：

  • Windows：Shadowsocks-Windows

  • macOS：ShadowsocksX-NG

  • Android：Shadowsocks for Android

  • iOS：Shadowrocket（需非大陆Apple ID购买）

4.2 添加配置并连接

输入你VPS的IP地址、端口、密码和加密方式。连接成功后，尝试访问 Google、YouTube 等网站，即可验证是否配置正确。

五、进阶玩法与实用建议

5.1 配置防火墙

确保你的VPS端口（如8388）已放通：

bash
ufw allow 8388

5.2 使用多用户、多端口配置

适合家庭共享或多设备使用，只需添加多个配置项即可。

5.3 使用加密和混淆插件（obfs）

提高隐蔽性，防止被GFW识别。安装 simple-obfs 插件并配置 plugin=obfs-local 即可。

5.4 定期更换端口和密码

增强安全性，避免被扫描封锁。

六、常见问题答疑

Q1：搭建费用高吗？

不高。一个基本VPS每月仅需$5左右，一年也就300元左右，比起一年好几百的VPN划算太多。

Q2：为什么连接不上？

  • 检查配置参数是否一致（密码、加密方式）

  • 检查防火墙是否开放端口

  • VPS是否被墙（可尝试更换IP）

Q3：可以多个设备同时使用吗？

可以。你只需在不同设备上安装客户端，并使用相同配置即可。

Q4：搭建服务会被封吗？

理论上可能，但只要注意使用隐蔽协议、不传播、不高调，基本不会有问题。建议定期更换端口与加密方式，保持低调。

FAQ

Shadowrocket 的分流规则如何优化视频播放？
通过分流规则，将流媒体域名匹配到高速节点，而其他流量走备用节点。保证视频播放流畅，同时节约带宽资源，提升整体网络速度和体验。
V2Ray 的 VMess over WebSocket 适合什么场景？
VMess over WebSocket 可以伪装成正常的 Web 流量，适用于被封锁较严格的网络环境。它通过将代理流量嵌入 WebSocket 协议中，使流量更难被检测和拦截。
QuantumultX 的 MITM 功能如何使用？
QuantumultX 的 MITM（中间人）功能可拦截 HTTPS 流量，用于修改或分析请求与响应。开启时需安装客户端生成的根证书，确保能正确解析加密数据。
VLESS + Reality 模式相比普通 TLS 有什么优势？
Reality 模式通过伪装握手和真实网站特征，提高流量隐蔽性，难以被 DPI 检测。TLS 模式更侧重加密和传输效率，Reality 更适合高封锁网络环境。
Netch 可以做什么？
Netch 是一款 Windows 平台的网络加速工具，支持基于 V2Ray、Socks5、Shadowsocks 等协议的代理。它可用于加速游戏、视频或网站访问，并能通过 TAP 模式实现系统级代理，广受游戏玩家喜爱。
Matsuri 的节点自动测速功能如何配置？
Matsuri 可以设置测速周期和节点列表，程序会自动测试延迟和速度并排序。用户可根据测试结果选择最佳节点，提高网络访问速度和使用体验。
如何配置 Sing-box 使用 DoH 加密 DNS？
在 dns 模块中添加类型为 doh 的上游，如 `https://1.1.1.1/dns-query`，并指定 bootstrap 地址。启用加密 DNS 可防止 DNS 泄露并加快国际解析速度，提高整体隐私保护。
Clash 节点延迟检测如何帮助分流？
Clash 会定期测试节点延迟，并结合策略组自动选择最优节点处理流量。避免访问高延迟或失效节点，提高网页浏览、视频播放和游戏速度及稳定性，优化整体网络体验。
VLESS 是否比 Trojan 更稳定？
两者都非常稳定，但 VLESS 在多平台兼容性上略优，而 Trojan 在 HTTPS 伪装和 TLS 安全性上更强。选择取决于服务器环境与具体需求，两者皆为高性能协议。
VLESS + XTLS 在高延迟网络中如何优化连接？
XTLS 模式通过减少握手和加密开销降低延迟。结合策略组和多节点切换，可在高延迟或不稳定网络环境下保证网页浏览、视频播放和游戏连接稳定，提高用户体验。
